# 微语 API 文档

**语言 / Language:** [中文](README.zh.md) | [English](README.md)

```bash
# 安装依赖
pnpm install
# 运行开发服务器
pnpm dev
# 构建静态文件
pnpm build
```

## 查看本地预览

`http://localhost:9018`
