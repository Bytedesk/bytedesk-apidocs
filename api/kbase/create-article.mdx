# 创建知识库文章

创建一篇新的知识库文章。

<RequestExample>

```bash cURL
curl --location --request POST 'https://api.weiyuai.cn/api/v1/knowledge/article/create' \
--header 'Authorization: Bearer YOUR_ACCESS_TOKEN' \
--header 'Content-Type: application/json' \
--data-raw '{
  "title": "如何重置密码",
  "content": "详细的密码重置步骤...",
  "category": "account",
  "tags": ["密码", "账号安全"],
  "status": "published"
}'
```

```python Python
import requests

url = "https://api.weiyuai.cn/api/v1/knowledge/article/create"
headers = {
    "Authorization": "Bearer YOUR_ACCESS_TOKEN",
    "Content-Type": "application/json"
}
payload = {
    "title": "如何重置密码",
    "content": "详细的密码重置步骤...",
    "category": "account",
    "tags": ["密码", "账号安全"],
    "status": "published"
}

response = requests.post(url, json=payload, headers=headers)
print(response.json())
```

```javascript JavaScript
const axios = require('axios');

const url = 'https://api.weiyuai.cn/api/v1/knowledge/article/create';
const headers = {
  'Authorization': 'Bearer YOUR_ACCESS_TOKEN',
  'Content-Type': 'application/json'
};
const data = {
  title: '如何重置密码',
  content: '详细的密码重置步骤...',
  category: 'account',
  tags: ['密码', '账号安全'],
  status: 'published'
};

axios.post(url, data, { headers })
  .then(response => console.log(response.data))
  .catch(error => console.error(error));
```

## 请求参数

<ParamField path="title" type="string" required>
  文章标题
</ParamField>

<ParamField path="content" type="string" required>
  文章内容（支持 Markdown 格式）
</ParamField>

<ParamField path="category" type="string" required>
  文章分类
</ParamField>

<ParamField path="tags" type="array">
  标签数组
</ParamField>

<ParamField path="status" type="string" default="draft">
  文章状态：`draft`（草稿）、`published`（已发布）、`archived`（已归档）
</ParamField>

<ParamField path="summary" type="string">
  文章摘要
</ParamField>

<ParamField path="coverImage" type="string">
  封面图片 URL
</ParamField>
