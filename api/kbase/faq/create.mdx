## 概述

创建一个新的常见问题（FAQ）条目。

  <ApiPlayground

## 接口说明

此接口用于创建新的 FAQ 条目。FAQ 用于存储常见问题和标准答案，可被智能客服自动引用回复。

## 请求头

```
Authorization: Bearer <access_token>
Content-Type: application/json
```

## 请求体

```json
{
  "question": "如何重置密码？",
  "answer": "您可以通过以下步骤重置密码：1. 点击登录页面的忘记密码...",
  "categoryUid": "category_123456",
  "tags": ["账号", "密码"],
  "priority": 1,
  "isEnabled": true
}
```

## 字段说明

### 请求字段

- `question` (必需): 问题内容
- `answer` (必需): 答案内容
- `categoryUid` (可选): 分类唯一标识符
- `tags` (可选): 标签数组
- `priority` (可选): 优先级，数字越大优先级越高，默认 0
- `isEnabled` (可选): 是否启用，默认 true

### FAQ 对象字段

- `uid`: FAQ 唯一标识符
- `question`: 问题内容
- `answer`: 答案内容
- `categoryUid`: 分类 UID
- `categoryName`: 分类名称
- `tags`: 标签数组
- `priority`: 优先级
- `isEnabled`: 是否启用
- `hitCount`: 命中次数
- `createdAt`: 创建时间
- `updatedAt`: 更新时间

## 使用场景

- 创建常见问题库
- 智能客服自动回复
- 知识库管理
- FAQ 匹配

## 注意事项

- 问题内容应简洁明确
- 答案内容应详细准确
- 可以设置多个相似问题指向同一个答案
- 支持富文本格式

```javascript JavaScript
const response = await fetch('https://api.weiyuai.cn/api/v1/faq', {
  method: 'POST',
  headers: {
```

```python Python
import requests

url = "https://api.weiyuai.cn/api/v1/faq"

  "Authorization": "Bearer your_access_token_here",
  "Content-Type": "application/json"
}
data = {
  "question": "如何重置密码？",
  "answer": "您可以通过以下步骤重置密码：1. 点击登录页面的忘记密码...",
  "categoryUid": "category_123456",
  "tags": ["账号", "密码"],
  "priority": 1,
  "isEnabled": True
}

response = requests.post(url, headers=headers, json=data)
result = response.json()
faq = result["data"]
```
