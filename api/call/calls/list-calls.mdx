# 呼叫列表

获取呼叫中心的呼叫记录列表。

<RequestExample>

```bash cURL
curl --location --request GET 'https://api.weiyuai.cn/api/v1/call/list?page=1&pageSize=20&status=completed' \
--header 'Authorization: Bearer YOUR_ACCESS_TOKEN'
```

```python Python
import requests

url = "https://api.weiyuai.cn/api/v1/call/list"
headers = {
    "Authorization": "Bearer YOUR_ACCESS_TOKEN"
}
params = {
    "page": 1,
    "pageSize": 20,
    "status": "completed"
}

response = requests.get(url, headers=headers, params=params)
print(response.json())
```

```javascript JavaScript
const axios = require('axios');

const url = 'https://api.weiyuai.cn/api/v1/call/list';
const headers = {
  'Authorization': 'Bearer YOUR_ACCESS_TOKEN'
};
const params = {
  page: 1,
  pageSize: 20,
  status: 'completed'
};

axios.get(url, { headers, params })
  .then(response => console.log(response.data))
  .catch(error => console.error(error));
```

## 查询参数

<ParamField query="page" type="number" default="1">
  页码
</ParamField>

<ParamField query="pageSize" type="number" default="20">
  每页数量
</ParamField>

<ParamField query="status" type="string">
  筛选呼叫状态：`ringing`、`connected`、`completed`、`failed`
</ParamField>

<ParamField query="startTime" type="string">

</ParamField>
