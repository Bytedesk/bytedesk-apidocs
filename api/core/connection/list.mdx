## 概述

查询用户的在线连接信息，包括设备类型、连接状态等。

  <ApiPlayground

## 接口说明

此接口用于获取当前用户的在线连接列表，支持分页查询。连接信息包括设备类型、操作系统、浏览器、最后活跃时间等。

## 请求头

```
Authorization: Bearer <access_token>
```

## 查询参数

- `page` (可选): 页码，默认为 0
- `size` (可选): 每页数量，默认为 10

## 字段说明

### 连接对象字段

- `uid`: 连接唯一标识符
- `userUid`: 用户唯一标识符
- `deviceType`: 设备类型（WEB, DESKTOP, MOBILE, TABLET）
- `os`: 操作系统
- `browser`: 浏览器
- `ipAddress`: IP 地址
- `status`: 连接状态（ACTIVE, INACTIVE）
- `lastActiveAt`: 最后活跃时间
- `createdAt`: 连接创建时间

## 使用场景

- 查看当前登录的设备
- 账号安全管理
- 异常登录检测
- 多设备管理

## 注意事项

- 只返回当前用户的连接信息
- 会在用户离线后保留一段时间的历史记录
- 支持强制下线某个连接

<RequestExample>

```bash cURL
curl -X GET "https://api.weiyuai.cn/api/v1/connection?page=0&size=10" \
```

```javascript JavaScript
const response = await fetch('https://api.weiyuai.cn/api/v1/connection?page=0&size=10', {
  method: 'GET',
```

```python Python
import requests

url = "https://api.weiyuai.cn/api/v1/connection"
headers = {
  "Authorization": "Bearer your_access_token_here",
  "Accept": "application/json"
}
params = {
  "page": 0,
  "size": 10
}

response = requests.get(url, headers=headers, params=params)
result = response.json()
connections = result["data"]["content"]
```
