## 概述

向指定的会话发送消息，支持文本、图片、文件等多种消息类型。

## 接口说明

此接口用于向指定的会话发送消息。消息类型包括文本、图片、语音、视频、文件、位置、卡片等。

## 请求头

```
Authorization: Bearer <access_token>
Content-Type: application/json
```

## 请求体

```json
{
  "threadUid": "thread_123456",
  "type": "text",
  "content": "消息内容",
  "extra": {}
}
```

## 字段说明

### 请求字段

- `threadUid` (必需): 会话唯一标识符
- `type` (必需): 消息类型
  - `text`: 文本消息
  - `image`: 图片消息
  - `audio`: 语音消息
  - `video`: 视频消息
  - `file`: 文件消息
  - `location`: 位置消息
  - `card`: 卡片消息
  - `event`: 事件消息
- `content` (必需): 消息内容，根据类型不同格式不同
- `extra` (可选): 扩展字段，用于存储额外的自定义数据

### 消息对象字段

- `uid`: 消息唯一标识符
- `threadUid`: 会话唯一标识符
- `senderUid`: 发送者用户唯一标识符
- `receiverUid`: 接收者用户唯一标识符（私聊时使用）
- `type`: 消息类型
- `content`: 消息内容
- `extra`: 扩展字段
- `status`: 消息状态（SENDING, SENT, READ, FAILED）
- `createdAt`: 消息创建时间

## 使用场景

- 即时通讯
- 客服对话
- 群组聊天
- 系统通知

## 注意事项

- 需要先加入会话才能发送消息
- 消息内容会经过敏感词过滤
- 文件、图片等需要先上传获取 URL
- 消息发送后会通过 WebSocket 推送给会话成员

<RequestExample>

```bash cURL
# 发送文本消息
curl -X POST "https://api.weiyuai.cn/api/v1/message/send" \
  -H "Authorization: Bearer your_access_token_here" \
  -H "Content-Type: application/json" \
  -d '{
    "threadUid": "thread_123456",
    "type": "text",
    "content": "你好，这是一条测试消息"
  }'

# 发送图片消息
curl -X POST "https://api.weiyuai.cn/api/v1/message/send" \
  -H "Authorization: Bearer your_access_token_here" \
  -H "Content-Type: application/json" \
  -d '{
    "threadUid": "thread_123456",
    "type": "image",
    "content": "https://example.com/image.jpg",
    "extra": {
      "width": 800,
      "height": 600
    }
  }'
```

```javascript JavaScript
// 发送文本消息
const response = await fetch('https://api.weiyuai.cn/api/v1/message/send', {
  method: 'POST',
  headers: {
    'Authorization': 'Bearer your_access_token_here',
    'Content-Type': 'application/json'

    type: 'text',
    content: '你好，这是一条测试消息'
  })
});

const result = await response.json();
const message = result.data;
```

```python Python
import requests

url = "https://api.weiyuai.cn/api/v1/message/send"
headers = {
  "Authorization": "Bearer your_access_token_here",
  "Content-Type": "application/json"
}

# 发送文本消息
data = {
  "threadUid": "thread_123456",
  "type": "text",
  "content": "你好，这是一条测试消息"
}

response = requests.post(url, headers=headers, json=data)
result = response.json()
message = result["data"]
```
