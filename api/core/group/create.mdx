## 概述

创建一个新的群组，支持设置群组名称、描述、头像等信息。

## 接口说明

此接口用于创建新的群组。创建者自动成为群组管理员。创建后可以邀请其他成员加入。

## 请求头

```
Authorization: Bearer <access_token>
Content-Type: application/json
```

## 请求体

```json
{
  "name": "群组名称",
  "description": "群组描述",
  "avatar": "https://example.com/avatar.png",
  "type": "PUBLIC",
  "maxMembers": 500,
  "needApproval": false
}
```

## 字段说明

### 请求字段

- `name` (必需): 群组名称，长度 1-50 字符
- `description` (可选): 群组描述，长度 0-200 字符
- `avatar` (可选): 群组头像 URL
- `type` (可选): 群组类型
  - `PUBLIC`: 公开群组
  - `PRIVATE`: 私有群组
  - `ORG`: 组织群组
- `maxMembers` (可选): 最大成员数，默认 500
- `needApproval` (可选): 是否需要审批加入，默认 false

### 群组对象字段

- `uid`: 群组唯一标识符
- `name`: 群组名称
- `description`: 群组描述
- `avatar`: 群组头像 URL
- `type`: 群组类型
- `maxMembers`: 最大成员数
- `needApproval`: 是否需要审批
- `memberCount`: 当前成员数
- `ownerUid`: 群主用户 UID
- `createdAt`: 创建时间

## 使用场景

- 创建项目讨论组
- 创建部门群组
- 创建兴趣群组
- 创建临时讨论组

## 注意事项

- 群组名称不能为空
- 群组类型创建后不能修改
- 创建者自动成为群主
- 群组创建后自动生成对应的会话

```javascript JavaScript
const response = await fetch('https://api.weiyuai.cn/api/v1/group', {
```

```python Python
import requests
```
