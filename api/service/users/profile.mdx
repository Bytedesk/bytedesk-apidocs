## 概述

获取当前登录用户的详细个人信息，包括基本资料和设置，需要身份验证。

## 接口说明

返回当前用户的完整个人信息，用于用户个人中心的信息展示。

## 请求头

```
Authorization: Bearer <access_token>
```



## 字段说明

- `id`: 用户唯一标识符
- `username`: 用户名
- `nickname`: 显示昵称
- `email`: 邮箱地址
- `mobile`: 手机号码
- `avatar`: 头像图片 URL
- `status`: 用户状态（ACTIVE: 正常, INACTIVE: 未激活, BANNED: 被禁用）
- `createdAt`: 账户创建时间

## 使用场景

- 用户个人中心展示
- 获取用户基本信息
- 验证用户身份状态
- 个人资料页面渲染

## 注意事项

- 需要有效的 Bearer Token
- 只能获取当前登录用户的信息
- 敏感信息可能会被过滤
