## 快速开始

要开始使用微语的AI对话功能，您需要：

1. **配置AI模型提供商** - 选择并配置您偏好的AI模型提供商
2. **设置多模态支持** - 启用文本、图片、视频、文件处理能力  
3. **接入知识库** - 可选择接入第三方知识库提升专业问答能力
4. **API集成** - 通过我们的REST API轻松集成到您的应用中

## API示例

### 发送文本消息到AI

```bash
curl -X POST https://api.weiyuai.cn/api/v1/ai/chat \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "message": "你好，我需要帮助",
    "type": "text",
    "provider": "qwen"
  }'
```

### 发送图片进行分析

```bash
curl -X POST https://api.weiyuai.cn/api/v1/ai/chat \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "message": "请分析这张图片",
    "type": "image",
    "image_url": "https://example.com/image.jpg",
    "provider": "openai"
  }'
```

## 集成指南

### 配置AI提供商

您可以通过管理后台配置多个AI提供商，系统会根据负载和可用性自动选择最优的提供商：

```javascript
// 客户端配置示例
const aiConfig = {
  providers: ['qwen', 'openai', 'deepseek'],
  defaultProvider: 'qwen',
  fallbackEnabled: true
}
```

### 多模态处理

系统自动识别输入类型并调用相应的AI模型：

```javascript
// 自动类型识别
const response = await fetch('/api/v1/ai/chat', {
  method: 'POST',
  headers: {
    'Authorization': 'Bearer YOUR_TOKEN',
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    input: file, // 可以是文本、图片、视频或文档
    autoDetectType: true
  })
})
```
