## 快速开始

要开始使用ByteDesk的AI对话功能，您需要：

1. **配置AI模型提供商** - 选择并配置您偏好的AI模型提供商
2. **设置多模态支持** - 启用文本、图片、视频、文件处理能力  
3. **接入知识库** - 可选择接入第三方知识库提升专业问答能力
4. **API集成** - 通过我们的REST API轻松集成到您的应用中

## API示例

### 发送文本消息到AI

```bash
curl -X POST https://api.weiyuai.cn/api/v1/ai/chat \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "message": "你好，我需要帮助",
    "type": "text",
    "provider": "qwen"
  }'
```

### 发送图片进行分析

```bash
curl -X POST https://api.weiyuai.cn/api/v1/ai/chat \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "message": "请分析这张图片",
    "type": "image",
    "image_url": "https://example.com/image.jpg",
    "provider": "openai"
  }'
```
