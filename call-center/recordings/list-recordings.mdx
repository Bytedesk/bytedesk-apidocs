# 录音列表

获取呼叫录音列表。

<RequestExample>

```bash cURL
curl --location --request GET 'https://api.weiyuai.cn/api/v1/recording/list?page=1&pageSize=20' \
--header 'Authorization: Bearer YOUR_ACCESS_TOKEN'
```

```python Python
import requests

url = "https://api.weiyuai.cn/api/v1/recording/list"
headers = {
    "Authorization": "Bearer YOUR_ACCESS_TOKEN"
}
params = {
    "page": 1,
    "pageSize": 20
}

response = requests.get(url, headers=headers, params=params)
print(response.json())
```

```javascript JavaScript
const axios = require('axios');

const url = 'https://api.weiyuai.cn/api/v1/recording/list';
const headers = {
  'Authorization': 'Bearer YOUR_ACCESS_TOKEN'
};
const params = {
  page: 1,
  pageSize: 20
};

axios.get(url, { headers, params })
  .then(response => console.log(response.data))
  .catch(error => console.error(error));
```

</RequestExample>

## 查询参数

<ParamField query="page" type="number" default="1">
  页码
</ParamField>

<ParamField query="pageSize" type="number" default="20">
  每页数量
</ParamField>

<ParamField query="callId" type="string">
  筛选指定呼叫的录音
</ParamField>

<ParamField query="startTime" type="string">
  开始时间（ISO 8601格式）
</ParamField>

<ParamField query="endTime" type="string">
  结束时间（ISO 8601格式）
</ParamField>

## 响应字段

<ResponseField name="recordings" type="array">
  录音记录数组
