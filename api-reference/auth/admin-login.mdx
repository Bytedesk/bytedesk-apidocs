## 接口说明

此接口用于使用 AccessToken 直接登录到管理后台，成功后会重定向到后台首页。

## 请求参数

- `accessToken` (必需): 有效的访问令牌

## 响应说明

- `302`: 重定向到管理后台
- `400`: 请求参数错误

## 使用场景

- 第三方系统集成
- 单点登录 (SSO)
- 免密登录场景

## 注意事项

- 此接口通常返回 HTTP 302 重定向
- AccessToken 需要预先通过其他方式获取
- 适用于网页端跳转场景

<RequestExample>

```bash cURL
curl --request GET \
  --url "https://api.weiyuai.cn/auth/login?accessToken=your_access_token_here" \
  --header "Accept: application/json"
```

```javascript JavaScript
const response = await fetch('https://api.weiyuai.cn/auth/login?accessToken=your_access_token_here', {
  method: 'GET',
  headers: {
    'Accept': 'application/json'
  }
});
```

```python Python
import requests

url = "https://api.weiyuai.cn/auth/login"
params = {
    "accessToken": "your_access_token_here"
}

response = requests.get(url, params=params)
```
