## 接口说明

当访问令牌过期时，可以使用刷新令牌获取新的访问令牌。

## 请求体

```json
{
  "refreshToken": "string"
}
```

## 响应示例

### 成功响应 (200)

```json
{
  "accessToken": "eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...",
  "expiresIn": 3600
}
```

### 错误响应 (401)

```json
{
  "error": 401,
  "message": "Invalid refresh token"
}
```

## 使用场景

- 访问令牌续期
- 自动刷新用户会话
- 保持用户登录状态

## 注意事项

- 刷新令牌通常有更长的有效期
- 建议在访问令牌过期前主动刷新
- 刷新令牌失效时需要重新登录

    "Content-Type": "application/json"
