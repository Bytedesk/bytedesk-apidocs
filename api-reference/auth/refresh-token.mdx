---
title: "POST 刷新令牌"
openapi: "POST /auth/token/refresh"
---

使用有效的刷新令牌获取新的访问令牌，用于延长用户会话时间。

## 接口说明

当访问令牌即将过期或已过期时，可以使用此接口通过刷新令牌获取新的访问令牌，避免用户重新登录。

## 请求体

```json
{
  "refreshToken": "string"
}
```

## 响应示例

### 成功响应 (200)

```json
{
  "accessToken": "eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...",
  "expiresIn": 3600
}
```

### 错误响应 (401)

```json
{
  "error": 401,
  "message": "Invalid refresh token"
}
```

## 使用场景

- 访问令牌续期
- 自动刷新用户会话
- 保持用户登录状态

## 注意事项

- 刷新令牌通常有更长的有效期
- 建议在访问令牌过期前主动刷新
- 刷新令牌失效时需要重新登录

<RequestExample>

```bash cURL
curl -X POST "https://api.weiyuai.cn/api/v1/auth/token/refresh" \
  -H "Content-Type: application/json" \
  -d '{
    "refreshToken": "your_refresh_token_here"
  }'
```

```javascript JavaScript
const response = await fetch('https://api.weiyuai.cn/api/v1/auth/token/refresh', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    refreshToken: 'your_refresh_token_here'
  })
});

const data = await response.json();
```

```python Python
import requests
import json

url = "https://api.weiyuai.cn/api/v1/auth/token/refresh"
headers = {
    "Content-Type": "application/json"
}
data = {
    "refreshToken": "your_refresh_token_here"
}

response = requests.post(url, headers=headers, json=data)
```
