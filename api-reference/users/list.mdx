## 接口说明

此接口用于分页获取系统中的用户列表，支持分页参数，需要管理员权限。

## 请求头

```
Authorization: Bearer <access_token>
```

## 查询参数

- `page` (可选): 页码，默认为 1
- `size` (可选): 每页数量，默认为 10

## 字段说明

- `users`: 用户列表数组
- `total`: 用户总数

### 用户对象字段

- `id`: 用户唯一标识符
- `username`: 用户名
- `nickname`: 显示昵称
- `email`: 邮箱地址
- `mobile`: 手机号码
- `avatar`: 头像图片 URL
- `status`: 用户状态
- `createdAt`: 账户创建时间

## 使用场景

- 管理后台用户管理
- 用户列表展示
- 用户数据分析
- 批量用户操作

## 注意事项

- 需要管理员权限
- 支持分页查询以提高性能
- 可能包含敏感信息，需要权限控制

```javascript JavaScript
const response = await fetch('https://api.weiyuai.cn/api/v1/users?page=1&size=10', {
  method: 'GET',
  headers: {
    'Authorization': 'Bearer your_access_token_here',
    'Accept': 'application/json'
  }
});

const data = await response.json();
const users = data.users;
const total = data.total;
```

```python Python
import requests

url = "https://api.weiyuai.cn/api/v1/users"
headers = {
    "Authorization": "Bearer your_access_token_here",
    "Accept": "application/json"
}
params = {
    "page": 1,
    "size": 10
}

response = requests.get(url, headers=headers, params=params)
data = response.json()
```
