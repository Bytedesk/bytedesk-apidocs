# 认证 API

这里包含用户认证相关的 API 端点，包括 AccessToken 免登、Bearer Token 刷新和用户注册登录等。

## 认证方式说明

- **AccessToken 免登**：用于网页跳转场景，通常返回 302 重定向
- **Bearer Token**：用于 REST API 调用，需在请求头携带 `Authorization: Bearer <token>`

## 接口列表

### 管理后台登录
使用 AccessToken 免登管理后台

### 刷新令牌
刷新 Bearer Token 以延长访问时效

### 用户注册
通过短信/邮箱验证码完成用户注册

### 用户登录  
使用用户名密码进行登录认证
