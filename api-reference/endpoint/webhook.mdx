Receive notifications when a new plant is added to the store. Configure your webhook endpoint to receive POST requests with plant data.

## Webhook Payload

<ParamField body="name" type="string" required>
  The name of the newly added plant
</ParamField>

<ParamField body="tag" type="string">
  Tag to specify the type of plant
</ParamField>

<ParamField body="id" type="integer" required>
  Unique identification number of the plant
</ParamField>

<ParamField body="timestamp" type="string" required>
  ISO 8601 timestamp when the plant was added
</ParamField>

## Response

Your webhook endpoint should return a `200` status code to acknowledge receipt.

<RequestExample>

```bash Example Webhook Payload
POST /your-webhook-endpoint
Content-Type: application/json

{
  "name": "Snake Plant",
  "tag": "low-maintenance",
  "id": 456,
  "timestamp": "2024-01-15T10:30:00Z"
}
```

</RequestExample>

<ResponseExample>

```json 200
{
  "status": "received"
}
```

</ResponseExample>
